# 飞影提示词与口播模板

## 使用原则

V1 每个商品只生成 1 条标准视频。提示词要稳定、可复制，优先保证商品还原和画面一致，不追求每条都强变化。

## 标准画面提示词

```text
竖屏9:16电商种草短视频，左右分屏构图，中间白色竖向分隔线。
左侧是一位亲和自然的年轻女性数字人，在真实[场景]中面对镜头口播，表情自然，动作轻微，穿着干净简洁。
右侧是手持[产品名称]的近景展示，商品主体清晰完整，尽量保持商品包装和外观与参考图一致，手部自然，真实光线，轻微镜头晃动。
整体风格真实、干净、自然光、电商产品推荐视频，不夸张，不卡通，不科幻。
口播内容围绕：[核心卖点]。
```

## 品类场景替换

| 品类 | 场景 |
| --- | --- |
| 生鲜食品 | 温室、菜市场、厨房、餐桌旁 |

| 运动户外 | 健身房、户外步道、运动装备区 |

## 负面提示词

```text
不要生成卡通风格，不要过度美颜，不要夸张表演，不要让商品变形，不要改变包装文字，不要多手指，不要手指穿过商品，不要模糊商品，不要遮挡商品主体，不要出现无关品牌，不要出现医疗功效暗示。
```

## 口播脚本模板

```text
最近[场景痛点]的人，可以看看这个[产品名称]。
它的[卖点1]比较适合[使用场景]，[卖点2]用起来更省心。
如果你平时在意[核心利益点]，[卖点3]这一点会很加分。
日常[使用方式]很顺手，想要省心一点的，可以先收藏看看。
```

## 示例

输入：

```text
产品名称：山野小青菜
核心卖点：根系完整；叶片厚实；清炒清甜
品类：fresh_food
```

口播：

```text
最近买菜怕不新鲜的人，可以看看这个山野小青菜。它根系比较完整，叶片也厚实，适合家里日常清炒或者做汤。入口是清甜的，不会一炒就软塌塌。想给家里备点省心蔬菜的，可以先收藏看看。
```

画面提示词：

```text
竖屏9:16电商种草短视频，左右分屏构图，中间白色竖向分隔线。
左侧是一位亲和自然的年轻女性数字人，在明亮温室中面对镜头口播，表情自然，动作轻微，穿着干净围裙。
右侧是手持山野小青菜的近景展示，小青菜根系完整、叶片清晰，商品主体完整，手部自然，真实光线，轻微镜头晃动。
整体风格真实、干净、自然光、电商产品推荐视频，不夸张，不卡通。
口播内容围绕：根系完整、叶片厚实、清炒清甜。
```

## 质检提示

生成后重点看三件事：

1. 商品外观是否仍像客户原图。
2. 手部是否自然，是否穿模或多指。
3. 口播是否只讲客户提供的卖点，没有编造功效。
