# 自定义文案校准测试资料

## 测试目的

验证 GUI 的“使用导入文案”模式能在飞影页面关闭“AI 自动生成”，填入自定义口播，并在成功校验后继续生成视频。

本测试只建议跑 1 条，会消耗 1 条真实飞影生成积分。

## GUI 推荐选择

- 商品来源：导入商品表
- 商品表：`products/custom-script-calibration.csv`
- 商品图：`products/images/IPAD-CUSTOM-SCRIPT-001.png`
- 人物策略：`自动匹配人物池`
- 文案策略：`使用导入文案`

## 商品信息

- SKU：`IPAD-CUSTOM-SCRIPT-001`
- 产品名称：`便携高清平板电脑`
- 核心卖点：`高清大屏；轻薄便携；学习办公娱乐都方便`
- 品类：`electronics`
- 表格中的图片文件名：`IPAD-CUSTOM-SCRIPT-001.png`
- 口播文案：

```text
这款便携高清平板电脑，屏幕清晰，机身轻薄，日常看课、处理资料和追剧都很顺手。出门放进包里也不占地方，想要一台简单好用的平板，可以重点看看。
```

## 预期结果

- GUI 批次详情能看到人物策略和文案策略。
- 最终确认框能看到当前策略。
- 如果飞影文案开关定位成功，任务会继续生成并下载视频。
- 如果飞影文案开关或文案回填校验失败，任务应停在 `failed_pre_submit`，不会继续外层视频提交。
